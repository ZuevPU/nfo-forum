# Чеклист ручного теста в VK DevTools

> **Автоматизация:** API-часть чеклиста запускается скриптом `npm run vk-checklist:ps1`.
> Шаги с VK Bridge (фото, iframe, push-доставка) — только вручную в DevTools.

## Подготовка
- [ ] Mini App открывается в VK DevTools без ошибок в консоли
- [ ] Backend доступен по `VITE_API_URL`
- [ ] `SKIP_VK_SIGN=false` в production

## Сценарии (10)

1. **Новый пользователь**
   - [ ] Welcome → Register → выбор трека → Home

2. **Главная**
   - [ ] Профиль, текущий блок, быстрые ссылки работают

3. **Расписание**
   - [ ] Табы дней переключают события
   - [ ] Фильтр «Мой трек / Все» работает

4. **Обмен опытом**
   - [ ] Создание вопроса
   - [ ] Клик по вопросу → деталь с ответами
   - [ ] Входящий вопрос → ответ

5. **Задания**
   - [ ] Список заданий, отправка ответа
   - [ ] Загрузка фото (VKWebAppUploadFiles)

6. **Чек-ин**
   - [ ] Сохранение состояния, реакция бота

7. **Рефлексия**
   - [ ] Ответ на вечерний вопрос

8. **Рейтинг**
   - [ ] Список участников трека

9. **Push / Cron**
   - [ ] `POST /api/cron/morning-greeting` с `X-Cron-Secret`
   - [ ] Deep link `#/checkin` открывает нужную панель

10. **Lifecycle**
    - [ ] Сворачивание/разворачивание приложения не ломает состояние
