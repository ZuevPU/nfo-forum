# Наполнение админки (ручная настройка)

После выката на Timeweb экраны работают, но без контента показывают дефолты или пустые списки. Настройте через **Админка** в приложении (аккаунт с правами администратора).

## 1. Вопросы (вечерняя рефлексия)

Вкладка **Вопросы** → тип «Рефлексия».

Для блока вечерней рефлексии:

| Поле | Значение |
|------|----------|
| `groupId` | Одинаковый у всех вопросов одного блока (например `evening-day1`) |
| `publishTime` | Время открытия (МСК), напр. `19:30` |
| `endTime` | Время закрытия, напр. `23:59` |
| `track` | Пусто = все треки, или конкретный трек |

Вопросы с одним `groupId` показываются участнику как один блок.

## 2. Задания

Вкладка **Задания**.

### «Познакомься с 3 участниками» (нетворкинг)

| Поле | Значение |
|------|----------|
| Нетворкинг | **Да** |
| Контактов | **3** |
| Авто-подтверждение | **Нет** |

Участник нажимает «Выполнить» → получает 3 контакта → отправляет фото/ответ на проверку.

### Обычные задания

- **Авто-подтверждение** — Да, если не нужна ручная проверка
- **Фото обязательно** — по необходимости
- **Трек** — пусто или конкретный трек

## 3. Настройки (чек-ин, НФО, пороги)

Вкладка **Настройки**.

### Чек-ин состояния

- **Треки** — отметьте треки, где чек-ин доступен
- **Слоты** — времена открытия, напр. `08:30, 13:15, 19:30`
- **Интервалы** — по строке: `08:00-12:00|Утро`, `12:00-18:00|День`
- **Эмоции** — список настроений (по одной на строку)
- **Подписи шкалы энергии** — «еле держусь» (0), «нормально» (5), «заряжен» (10)
- **Заголовки** — title, subtitle, подписи полей

### День НФО

- Вопрос дня, факторы (по строке), заголовки панели
- Время публикации и баллы

### Пороги уровня рефлексии

Пороги баллов рефлексии (не рейтинга трека): **0, 55, 110, 164, 218** по умолчанию.

| Уровень | Название | Порог |
|---------|----------|-------|
| 1 | Начал задумываться | 0 |
| 2 | Удерживаю важное | 55 |
| 3 | Соединяю опыт | 110 |
| 4 | Вижу картину целиком | 164 |
| 5 | Строю свою траекторию | 218 |

### Система баллов

Вкладка **Настройки** → блок **«Система баллов»** — справочная таблица с редактируемыми лимитами.

**Два счётчика:**
- **Рейтинг трека** (`user.points`) — только задания форума после одобрения
- **Уровень рефлексии** (`reflectionPoints`) — рефлексия, чек-ин, диагностика, НФО дня, озарения, обмен

**Максимумы по секциям (дефолт):**
- Рефлексия — 228 б.
- Обмен — 45 б. (5 вопросов × 3 + 10 ответов × 3)
- Задания — 205 б. (справочно; фактические баллы задаются в карточке задания)
- Рефлексия + обмен — 273 б.

**Ключевые правила:**
- Чек-ин: эмоция 2 + энергия 2 + комментарий 3 (опционально)
- Обмен: 3 б. за опубликованный вопрос (макс. 5), 3 б. за ответ (макс. 10)
- Диагностика: 10 за прохождение + 5 за комментарий к профилю
- НФО вечер: thesis 5 + understanding 10 + factors 5 + extra 5 (опц.)
- Озарения: 3 б. × до 10 записей (экран «Вопросы»)

**Озарения** настраиваются только через лимит `insight` в таблице; UI — у участников на экране вопросов.

### Фокус дня

Заголовок задания дня на главной.

## 4. Расписание и события

Вкладка **Расписание** — блоки программы с временем, местом, треком.

Фильтр **«Общий трек»** на экране расписания показывает события без привязки к треку.

## 5. Проверка после наполнения

1. Главная — карточки «Активные вопросы», «Активные задания», чек-ин
2. `/tasks` — «Выполнить» открывает задание без дёргания экрана
3. Нетворкинг — 3 контакта в задании «Познакомься с 3»
4. Админка → Выгрузки — CSV/XLSX скачиваются
5. Рейтинг — пороги уровней из настроек
