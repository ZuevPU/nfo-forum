## 1. Шапка (header)

**Сейчас:** фиолетовый блок с именем и треком обрывается резко, карточка профиля висит отдельно под ним на белом фоне.

**Нужно:** карточка участника (аватар + имя + трек + баллы) должна быть **внутри** синего градиентного блока, не снаружи.

```
Цвет фона шапки: linear-gradient(135deg, #3b4ec0, #4f5ed4)
Карточка профиля внутри шапки: background: rgba(255,255,255,0.14), border-radius: 10px
Имя: font-size 14px, font-weight 700, color #fff
Бейдж трека: background rgba(59,191,160,0.28), border 1px solid rgba(59,191,160,0.5), color #3bbfa0
Баллы: background #3bbfa0, color #fff, border-radius 6px, padding 3px 8px
```

## 2. Блок «Фокус дня»

**Сейчас:** зелёный (#1a7a5e или похожий) — не из палитры проекта.

**Нужно:** заменить цвет фона на бирюзовый из палитры.

```
Фон блока: #3bbfa0
Текст заголовка «ФОКУС ДНЯ»: color #fff, font-size 11px, font-weight 700,
  text-transform uppercase, letter-spacing 0.08em
Текст вопроса: color #fff, font-size 16px, font-weight 700
Border-radius: 12px
Отступы: padding 14px 16px
```

---

## 3. Иконки активностей

**Сейчас:** эмодзи на одинаковом светлом квадрате — все одного цвета, нет визуального разделения.

**Нужно:** у каждого раздела своя цветная подложка под иконкой.

```
📅 Что далее / Программа   → background #e8eaff  (голубой)
💬 Активные вопросы        → background #e2f8f3  (бирюзовый)
⭐ Активные задания         → background #fff5e8  (жёлтый)
💡 Активный обмен опытом   → background #e2f8f3  (бирюзовый)
🧠 Уровень рефлексии       → background #f2e8ff  (фиолетовый)

Размер подложки: 36x36px, border-radius: 9px
```

## 4. Заголовок раздела «АКТИВНОСТИ»

**Сейчас:** крупный капс с большим отступом сверху — визуально тяжело.

**Нужно:** облегчить.

```
font-size: 10px
font-weight: 700
color: #888
text-transform: uppercase
letter-spacing: 0.08em
margin-top: 8px
margin-bottom: 6px
```

## 5. Карточки активностей

**Сейчас:** нет чёткой границы между карточками, разделитель едва заметен.

**Нужно:** каждая карточка — отдельный белый блок с лёгкой тенью.

```
background: #ffffff
border-radius: 12px
padding: 10px 14px
margin-bottom: 6px
box-shadow: 0 1px 4px rgba(0,0,0,0.06)
```

---

## 6. Нижняя навигация

**Сейчас:** активная вкладка выделена только цветом иконки.

**Нужно:** активная вкладка — синяя подпись.

```
Активная вкладка: цвет подписи #3b4ec0, font-weight 700
Неактивная вкладка: цвет подписи #aaa, font-weight 400
```

## 7. Персонаж — добавить на два экрана

### 7а. Экран приветствия (онбординг)

Добавить изображение персонажа (файл `персонаж.png` — прозрачный фон) **над заголовком «Привет!»**.

```
Размер: width 120px, height auto
Расположение: по центру, margin-bottom 10px
Фон экрана: linear-gradient(160deg, #3b4ec0 0%, #4f5ed4 50%, #6b5ecf 100%)
```

### 7б. Пустые состояния

Когда в разделе нет контента (0 вопросов, 0 заданий и т.д.) — показывать персонажа с фразой.

```
Размер персонажа: width 80px, height auto
Расположение: по центру экрана
Фраза под персонажем: color #888, font-size 13px, text-align center

Тексты по разделам:
- Задания: «Задания скоро появятся — следи за уведомлениями»
- Вопросы: «Вопросов пока нет. Загляни позже»
- Обмен опытом: «Пока никто не задал вопрос. Будь первым!»
```

### 7в. После выполнения задания

После нажатия «Отправить» / «Выполнено» — экран подтверждения с персонажем.

```
Персонаж: width 90px, по центру
Заголовок: «Отлично!» color #0f6e56, font-size 18px, font-weight 700
Подзаголовок: «+[N] баллов начислено» color #3bbfa0, font-size 14px
Фон блока подтверждения: #e2f8f3, border-radius 16px, padding 20px
```
